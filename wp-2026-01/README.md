# WP 2026.01 — The Accounting for Value

Aplicação do framework Penmaniano (FSAV Cap 5) a três arquétipos brasileiros:
MDIA3 (Case 1), VIVA3 (Case 2), RADL3 (Case 3).

## Estrutura

- `src/` — código Python que produz tabelas e cálculos do paper
- `data/` — CSVs canónicos (fonte única de verdade)
- `notes/` — notas de leitura Penman
- `paper/` — versões do paper em Word
